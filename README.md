# Shop Manager

Uma **loja online fictícia** feita com **JavaScript puro** (sem frameworks)!

## **Funcionalidades**

✅ **CRUD de produtos** (Criar, Editar, Listar)  
✅ **Carrinho dinâmico** - Adiciona/remove com 1 clique  
✅ **Cálculo automático** do total  
✅ **localStorage** - Tudo salvo no navegador  
✅ **Edição avançada** com formulário pré-preenchido  
✅ **Design responsivo** - Funciona no mobile  
✅ **Toggle visual** - Produtos selecionados destacados  

## **Como usar**

1. **Adicione produtos** (nome, preço, categoria)
2. **Clique em 🛒** para adicionar/remover do carrinho  
3. **Edite com ✏️** - Formulário pré-preenchido!
4. **Total atualiza** automaticamente

## **Tech Stack**

- JavaScript (ES6+)
- localStorage API
- CSS Grid + Flexbox
- Design System moderno
